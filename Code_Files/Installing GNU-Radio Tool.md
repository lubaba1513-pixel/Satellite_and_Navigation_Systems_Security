###### \###Installing GNU-Radio Tool###



1. Update System Packages



Command:

sudo apt update -y



Purpose:

* Updates system repositories
* Ensures latest packages are available



**2.** Upgrade System

\---

Command:
sudo apt upgrade



Purpose:
Upgrades installed packages
Improves system performance and stability



**3.** Install GNU Radio

\---

Command:
sudo apt install gnu-radio



Purpose:
Installs GNU Radio tool
Used for simulating satellite and wireless signals



**4.** Launch GNU Radio Companion

\---

Command:
gnuradio-companion



Purpose:
Opens graphical interface
Used to design signal flowgraphs




Kali Linux is used
